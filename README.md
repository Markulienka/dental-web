# Setup Project

1. Clone the repo
2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Setup environment variables:
   ```bash
   cp .env.secrets-example .env
   ```
4. Start development server:
   ```bash
   pnpm dev
   ```

5. Open `http://localhost:3000` and follow on-screen instructions to create your first admin user.
